## VAE

Visual Access and Exposure (VAE) 이론은 주어진 공간의 특정 위치에서 다른 위치를 얼마나 볼 수 있는지(access), 그리고 다른 위치에 얼마나 노출되는지(exposure)를 기반으로 공간을 분석하는 방법이다. 2015년 1학기에 건축 공간 분석학 수업을 들으며 다음의 실험을 진행하였다.
